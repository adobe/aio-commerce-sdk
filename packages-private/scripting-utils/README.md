# `@aio-commerce-sdk/scripting-utils`

Utility functions for Adobe Commerce SDK scripts and build tools.

This package provides common utilities for working with project structure (finding files and directories), error handling, and YAML processing. It's designed to be used internally by other Adobe Commerce SDK packages and build scripts.

## Installation

```bash
pnpm add @aio-commerce-sdk/scripting-utils
```
