# Breaking Changes

> All unreleased changes are planned for the next major release together with other accumulated breaking changes.

## [Unreleased]

### Breaking Changes (planned)

#### ky v2 hook signature change

The SDK uses [ky](https://github.com/sindresorhus/ky) as its HTTP client. ky v2 introduces breaking changes to hook signatures. See the [ky v2 release notes](https://github.com/sindresorhus/ky/releases/tag/v2.0.0) for details and the migration guide.
